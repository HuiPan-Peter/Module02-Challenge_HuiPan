Code Helper link: https://github.com/HuiPan-Peter/Code-Helper-Project

Technologies Used: - bcrypt - express - express-session - express-handlebars - connect-session-sequelize - mysql2 - sequelize

Usage Instructions: 1. Clone the GitHub repository: `git clone repository-link`. 2. Navigate to the project directory: `cd code-helper`. 3. Install dependencies: `npm install`. 4. Configure the `.env` file with your database credentials. 5. Initialize the database: `npm run db:init`. 6. Seed the database: `npm run seed`. 7. Launch the server: `npm start`.
